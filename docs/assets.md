# Asset Guide

All current assets are original placeholders produced for this project from the attached concept mood only. They do not copy real logos, official rule text, manufacturer designs, team liveries, or WEC branding.

Runtime assets are listed in `src/assets/manifest.ts`. Replace assets by keeping the same manifest keys and updating the file path and dimensions.

Recommended replacements:

- Background/key art: WebP, 1600px wide or smaller.
- Car and crew sprites: PNG/WebP with clean silhouettes.
- UI icons: SVG.
- Audio: OGG with MP3 fallback.
