# Rules Research

No official FIA/WEC rule text was required to implement this MVP because the requested gameplay rule is self-contained: two wheel-gun operators work one side, then cross to the other side. The game avoids official series names, logos, real manufacturers, and team liveries in shipped assets.

If a future realism mode needs exact championship regulation language, research should be added here with source URLs, access dates, and a short explanation of whether the rule is implemented literally or adjusted for arcade play.
